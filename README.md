游戏介绍

《Sandbox》是一款基于 Python 和 Pygame 开发的 2D 沙盒游戏，灵感源自经典沙盒游戏，并加入了独特的“塔拉瑞安”地形生成算法。

核心特性

· 无限生成的世界：每次开始新游戏都能探索独一无二的地图。
· 柏林噪声地形：自然的地表起伏、洞穴和生物群系。
· 完整的方块与物品系统：数百种方块可供建造，工具、武器一应俱全。
· 昼夜循环与怪物 AI：白天探索，夜晚抵御怪物的袭击。
· 完善的存档机制：随时保存和加载你的世界。
· 虚拟控制支持：在手机端也能流畅操作（跨平台兼容！）。

下载与运行

获取游戏

· 通过 GitHub Releases 下载最新版本的可执行文件或源码包。
· 或者直接克隆本仓库，运行 sandbox_game-latest.py（需安装 Python 3.8+ 和 Pygame）。

运行要求

· Python 3.8 或更高版本
· Pygame 2.0 或更高版本（安装命令：pip install pygame）

快速启动

```bash
git clone https://github.com/LongTeam-Studio/LongTeam-Studio.github.io.git
cd LongTeam-Studio.github.io
python sandbox_game-latest.py
```
